Operadores do JS

Parte 1:

Principais famílias de operadores em JS:

  • Aritméticos
  • Atribuição
  • Relacionais
  • Lógicos
  • Ternário

Aritméticos:

  • [+]: 2 + 3 = 5;
  • [-]: 7 - 4 = 3;
  • [*]: 3 * 4 = 12;
  • [/]: 18 / 9 = 2;
  • [%]: 5 % 2 = 1;
  • [**]: 3 ** 2 = 9;

Atribuição:

Operadores para atribuir valores a uma variável, sendo os principais "=" para atribuição simples ou atribuição com operação, exemplo: +=, -=, *=. Também a o de incremento "++" e decremento "--".

Parte 2:

Relacionais:

  • ">" Maior
  • "<" Menor
  • ">=" Maior ou igual
  • "<=" Menor ou igual
  • "==" Igual
  • "!=" Diferente

Lógicos:

  • "!" Negação
  • "&&" Conjunção, uma espécie de "e" lógico
  • "||" Disjunção, uma espécie de "ou" lógico

Ordem de execução: primeiiro o "!", depois o "&&", por fim o "||".

Ternário:

Possuí os símbolos "?" e ":", qual compõem uma expressão de três blocos na seguinte estrutura: Teste lógico ? True : False

Exemplo: Média >= 7.0 ? "Aprovado" : "Reprovado"
