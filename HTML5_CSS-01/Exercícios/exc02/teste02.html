OBS1: "hr" entre os sinais é uma Tag para criar uma linha horizontal

OBS2: "br" entre os sinais é uma Tag de quebra de linha, semelhante à barra dupla do portugol

OBS3: a letra "r" no final de ambas as Tags refere-se à "Row", que, neste contexto, significa "linha". Sendo assim, a Tag "hr" significa "Horizontal Row"(Linha Horizontal), enquanto a Tag "br" significa "Break Row"(Quebra de linha).
