Tecnologias Client Side = Aquelas que funcionam na máquina do cliente, como as linguagens HTML, CSS e JavaScript por exemplo.

Tecnologias Server Side = Aquelas que funcionam nos equipamentos dos servidores, como as linguagens php e python por exemplo.

Front-end = Quem utiliza principalmente tecnologias Client Side para desenvolver sites que operem na máquina do cliente.

Back-end = Quem utiliza principalmente tecnologias Server Side para desenvolver e administrar servidores e sites que utilizem tecnologias Server Side.

Full stack = Desenvolve tanto com Client Side quanto Server Side.
