→ Texto marcado

É possível marcar textos utilizando a tag <mark>, valendo ressaltar o fato de esta ser uma tag de significado.

Segundo exemplo de marcação de de texto, desta vez, um texto marcado com uma cor distinta da padrão.

→ Texto grande e pequeno

Estamos criando um exemplo de texto grande e um exemplo de texto pequeno neste parágrafo.

→ Texto deletado

Podemos marcar um texto como excluído para indicar que ele deve ser lido, mas não considerado.

→ Texto inserido

Podemos marcar um texto como inserido para dar uma ênfase e indicar que ele foi adicionado depois.

Támbem é possível adquirir um texto riscado e um texto sublinhado utilizando, respectivamente, as tags <s> e <u>, entretanto, estas são tags não semânticas.

→ Texto sobrescrito

Recomendado para utilizar em situações de textos "altos", como expoentes, por exemplo: 25x107+x56.

→ Texto subscrito

Recomendado para casos de textos "baixos", como índices, por exemplo: NH4OH ou H2O.
